# 🐍 Snake Game
My first Python project — a recreation of the classic Snake game from old Nokia phones, built with pygame.  
## **How to play**
- Use the arrow keys to move the snake
- Eat the red squares to grow and score points
- Avoid hitting the walls or yourself
- Press R to restart, ESC to quit

## **How to run it**

Install Python
Install pygame: pip install pygame
Run: python snake_game_en.py

## **What I learned**
This project helped me practice loops, conditionals, lists, and basic event handling in Python.
